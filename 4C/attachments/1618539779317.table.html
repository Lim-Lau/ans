配置项说明

配置描述

备注

spring.datasource.druid.db-type

数据库类型

值: postgresql

/

spring.datasource.druid.url

数据库地址URL

值: jdbc:postgres_jts://IP:5432/survey-assistant

将IP数据库IP地址修改crs为使用数据库实例名称

spring.datasource.druid.username

数据库用户名

默认：postgres值：修改为数据用户名

/

spring.datasource.druid.password

数据库密码

默认：postgres值：修改为数据库密码

/

spring.datasource.druid.driver-class-name

驱动名称

值：org.postgis.jts.JtsWrapper

/

spring.datasource.druid.filters

过滤机制

值：stat,wall,log4j2

/

spring.datasource.druid.initial-size

初始化时建立物理连接的个数

值：10

/

spring.datasource.druid.min-idle

最小连接数

值：1

/

spring.datasource.druid.max-active

最大活动连接数

值：50

/

spring.datasource.druid.max-wait

超时等待

值：60000

毫秒

spring.datasource.druid.time-between-eviction-runs-milli

关闭空闲连接的检测时间间隔

值：60000

毫秒

spring.datasource.druid.min-evictable-idle-time-milli

连接的最小生存时间

值：300000

毫秒

spring.datasource.druid. validation-query

校验sql

值：SELECT 'CRS'

/

spring.datasource.druid.test-while-idle

申请连接时检测空闲时间

默认：false值：true

/

spring.datasource.druid.test-on-borrow

申请连接时直接检测连接是否有效

默认：false值：true

/

spring.datasource.druid.test-on-return

数据源配置归还连接时检测连接是否有效

默认：false值：false

/

spring.datasource.druid.pool-prepared-statements

开启PSCache

默认：false值：false

/

spring.datasource.druid.max-pool-prepared-statement-per-connection-size

设置PSCache值

值：20

/

spring.datasource.druid.web-stat-filter.enabled

web统计筛选器是否开启

值：true

/

spring.datasource.druid.web-stat-filter.url-pattern

web统计筛选器路径

值：/*

/

spring.datasource.druid.web-stat-filter.exclusions

web统计筛选除外路径

值：/druid/*,*.js,*.gif,*.jpg,*.png,*.css,*.ico

/

spring.datasource.druid.stat-view-servlet.enabled

状态视图是否开启

值：true

/

spring.datasource.druid.stat-view-servle.url-pattern

状态视图servlet路径

值：/druid/*

/

spring.datasource.druid.stat-view-servle.reset-enable

状态视图servlet:重启应用

值：true

/
